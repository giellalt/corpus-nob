Kristuskransen Kristusen gievlie
Hvilestedet Lïegkedimmie-sijjie
Når du holder i stillhetsperlene kan du hvile i alt som er. Datne “manne-gearketjem” steereminie jïh dïsse vuartesjeminie.
Gud ser deg og tar imot deg. Du kan bare være. Gæjka seammalaakan goh Jupmele vuajna datne gïjkeminie.
Gud er hos deg uten krav. Datne leah Jupmelen læjhka-maana.
Du trenger ingen ord, du kan bare være stille. Daelie maahtah boelhketjem åajaldehtedh eah gaajhkesh iktesth vuejnieh man tjaebpie datne leah.
Kanskje du i fantasien kan finne et sted som du kan legge inn i stillhetsperlene som minner deg om den følelsen. Guktie Jupmele aavoedi gosse vööjni man tjaebpie gaajhke lea maam satne sjugniedi, naemhtie Jupmele datnine aavode.
